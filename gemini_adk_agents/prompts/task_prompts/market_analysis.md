# Market Analysis Task Prompt

## Task
Analyze current market conditions for: {market_or_sector}

## Analysis Areas

1. **Market Overview**
   - Current index levels
   - Daily/weekly performance
   - Market breadth
   - Volume analysis

2. **Sector Performance**
   - Top performing sectors
   - Underperforming sectors
   - Sector rotation patterns

3. **Key Movers**
   - Top gainers
   - Top losers
   - High volume stocks
   - New highs/lows

4. **Sentiment Analysis**
   - Market mood (bullish/bearish/neutral)
   - Fear/greed indicators
   - Institutional activity
   - Retail participation

5. **Outlook**
   - Short-term view (1 week)
   - Medium-term view (1 month)
   - Key levels to watch
   - Potential risks

## Output Format

Provide analysis with:
- Timestamp of analysis
- Data sources
- Confidence level
- Key actionable insights
